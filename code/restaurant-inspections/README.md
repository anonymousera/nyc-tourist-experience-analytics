# Restaurant Inspections Module

This module processes NYC restaurant inspection records and generates dining-quality indicators for neighborhood tourism analysis.

## Key Files

- `NYCInspection.java`
- `NYCInspectionMapper.java`
- `NYCInspectionReducer.java`
- `build.sh`: local build helper
- `commands.txt`: run command history
- `nyc-inspection.jar`: packaged artifact

## Sample Data and Artifacts

- `input/`: sample input files
- `output/`: sample output files
- `screenshots/`: execution snapshots

## Run Notes

- Build and execute in Hadoop environment.
- Validate on the sample files before full execution.
- Keep only representative output subsets in source control.
